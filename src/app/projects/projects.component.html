QFDCenter 2017 (In-progress)

The owner of QFDCenter were in need of better methods to conduct tests on users who were part of his tasting of food program. So to solve this problem 2 other people and I set out to try and create a better and more solid platform for the user to participate in tests and also a better and more functional space for the owner to create these tests. Meanwhile the site should be able to function similar to that of a mobile application and allow for offline submissions during flight on an airplane.

Derailed 2016 Student Project

This semester were about audio-visual experiments and so we set out to create a game in Unity and controlling it using the platform Airconsole.com on a mobile phone. Our game ended up being a co-opexperience with 4 players and a train that you have to keep alive for the journey by completing simple mini-games on the phone while the game is running on the screen/TV.

Drummute 2016 Student Project

The theme of this semester project was to create an electronic product that would work with sound. This resulted in a product that was intended to be used by drummers to allow them to have a portable drumkit. The drumkit functioned through pressure sensors to detect hit, location and force.

Music Creation Through Image Processing 2015 Student Project

This semester, the assignment was to work with image processing. My group and I ended up creating an application that could track your hand and react to gestures to control a program we had made to create simple musical tones. The program was also capable of controlling the computer.
